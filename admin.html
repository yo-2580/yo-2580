כיצד המערכת עובדת?
1. לחצו על "הוסף קישור חדש" ומלאו כותרת וכתובת אתר (URL).
2. לחצו על כפתור השמירה הכחול. המערכת תשלח בקשה מאובטחת לשרתי GitHub.
3. שרתי GitHub יריצו את תהליך השמירה (Action) ויעדכנו את קובץ הנתונים.
4. שימו לב: לוקח ל-GitHub כדקה לעדכן את האתר בפועל.
